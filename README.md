## 평가지표
> 1️⃣2️⃣3️⃣4️⃣5️⃣
### 난이도
- 1️⃣ : 눈으로 봐도 풀린다
- 2️⃣ : 생각할 조건이 있다
- 3️⃣ : 생각치 못한 조건이 있으나, 정답을 맞췄다
- 4️⃣ : 정답을 모르겠으나, 해답을 보니 이해가 된다
- 5️⃣ : 해답을 봐도 모르겠다
### 중요도
- 1️⃣ : 공장에서 찍은거 같은 문제
- 2️⃣ : 알고리즘 기본 유형 문제
- 3️⃣ : 알고리즘 응용 문제 
- 4️⃣ : 처음 보는 유형
- 5️⃣ : 두고두고 연습하면 좋을 문제
### 분노지수
- 1️⃣ : 쉽고 짧다
- 2️⃣ : 어려우나 짧다
- 3️⃣ : 쉬운데 길다
- 4️⃣ : 어렵고 길다
