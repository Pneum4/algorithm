[문제 설명]

그림판은 정사각형 모양을 가지고 N * N 개의 셀들로 구성되어 있다. N은 2K으로 표현되는 정수이다. (2 ≤ K ≤ 9)

 

[Fig. 1]는 N = 8, K = 3일 때 그림판의 예이다.

셀 안에 (r, c)는 셀이 r번째 행과 c번째 열에 위치하는 셀을 의미한다.

행과 열의 순서는 0부터 시작한다. (0 ≤ r, c ≤ N – 1)

 

각각의 셀은 색을 의미하는 0 또는 1의 값을 가진다.

 

그림판은 인코딩 또는 디코딩을 할 수 있다.

인코딩을 하면 각각의 셀들의 값들이 문자열로 변환되고 디코딩을 하면 문자열로부터 각각 셀들의 값들이 복원된다.

 

다음과 같은 규칙으로 인코딩한다. (‘x’는 문자 x를 의미한다.)

① - 모든 셀들의 값이 0이면 ‘0’으로 표시한다. 모든 셀들의 값이 1이면 ‘1’로 표시하고 끝난다.

② - 과정 ①에서 해당하지 않으면 가로 방향, 세로 방향으로 각각 균등하게 2등분하여 전체 영역을 4등분한다.

③ - 과정 ②에서 나눈 4등분 영역들을 왼쪽 위, 오른쪽 위, 왼쪽 아래, 오른쪽 아래 순서로 각각 과정 ①을 반복하고 그 결과들을 연속적으로 붙인다. ([Fig. 2] 참조)

④ - 과정 ③에서 나오는 결과를 ‘(’와 ‘)’로 감싼다.

 

[Fig. 3-1] ~ [Fig. 3-3]은 그림판을 인코딩한 예들이다. “”는 문자열을 의미한다.

굵은 선의 테두리는 인코딩할 때 영역이 분할된 것을 의미하고 분할된 영역 안에는 같은 값을 가지는 셀들만 있다.

 

디코딩 과정은 인코딩 과정을 반대로 진행하는 것이다.

 

그림판에서 점을 찍을 수 있다. 점은 색과 크기를 가지고 있다.

(r, c)에 크기가 mSize인 점을 찍게 되면 (r, c)와 거리가 mSize 미만인 셀에 대해서 점이 가지는 색의 값으로 셀들의 값들이 변경된다.

두 셀 (rA, cA)와 (rB, cB) 간의 거리는 |rA – rB| + |cA – cB|이다.

 

[Fig. 4-1] ~ [Fig. 4-4]는 점을 찍는 예들이다.

[Fig. 4-1]는 (2, 2)에 점을 찍기 전이고 [Fig. 4-2], [Fig. 4-3], [Fig. 4-4]는 색이 1이고 크기가 각각 1, 2, 3인 점을 찍은 후이다.

오렌지 색의 셀이 점을 찍는 위치이고 노란 색의 셀이 점을 찍고 난 후 값이 변경된 셀들이다.

 

그림판에서 어떤 색으로 페인트칠을 할 수 있다.

한 셀이 페인트칠되면 그 페인트칠이 된 셀로부터 상하좌우로 인접하고 같은 색의 값을 가지는 셀들도 페인트칠이 된다.

페인트칠이 된 모든 셀들의 값들은 페인트칠되는 색의 값으로 변경된다.

 

[Fig. 5-1] ~ [Fig. 5-3]을 페인트칠을 하는 예들이다.

[Fig. 5-1]는 (2, 2)에 페인트칠을 하기 전이고 [Fig. 5-2]와 [Fig. 5-3]은 페인트칠을 하는 색이 각각 1, 0일 때의 결과들이다.

오렌지 색의 셀이 페인트칠을 하는 위치이고 노란 색의 셀이 페인트칠을 한 후 값이 변경된 셀들이다.

 

이와 같은 그림판을 편집하는 프로그램을 작성하라.

 

아래 API 설명을 참조하여 각 함수를 구현하라.

※ 아래 함수 signature는 C/C++에 대한 것으로 다른 언어에 대해서는 제공되는 Main과 User Code를 참고하라.

아래는 User Code 부분에 작성해야 하는 API 의 설명이다. 

void init(int N, int L, char mCode[])

테스트 케이스에 대한 초기화 함수.

 

그림판은 N * N개의 셀로 구성되어 있다. 길이가 L인 문자열 mCode로부터 디코딩을 하여 각 셀들의 값들을 복원한다.

 

인코딩과 디코딩하는 과정은 본문 설명을 참조하라.

 

N은 2K으로 표현되는 정수이다. (2 ≤ K ≤ 9)

 

Parameters
  N : 그림판의 한 변의 길이 (4 ≤ N ≤ 512)
  L : 문자열 mCode 길이 (1 ≤ L ≤ 200,000)
  mCode : 인코딩된 문자열

int encode(char mCode[])

그림판의 각 셀들의 값들을 인코딩하고 인코딩된 문자열을 mCode 저장하고 그 길이를 반환한다.

 

인코딩된 문자열의 길이는 200,000을 넘지 않음을 보장한다.

 

예로, 인코딩한 결과가 “(0010)”인 경우 mCode[0] = ‘(’, mCode[1] = ‘0’, mCode[2] = ‘0’, mCode[3] = ‘1’, mCode[4] = ‘0’, mCode[5] = ‘)’을 저장하고 6을 반환하면 된다.

 

Parameters
  mCode : 인코딩된 문자열을 저장할 배열

Returns
  인코딩된 문자열의 길이

void makeDot(int mR, int mC, int mSize, int mColor)

(mR, mC)에 크기가 mSize이고 색이 mColor인 점을 찍는다.

 

점을 찍는 과정은 본문 설명을 참조하라

 

Parameters
  mR : 점을 찍고자 하는 셀의 행의 위치 (0 ≤ mR ≤ N - 1)
  mC : 점을 찍고자 하는 셀의 열의 위치 (0 ≤ mC ≤ N - 1)
  mSize : 점의 크기 (1 ≤ mSize ≤ 5)
  mColor : 점의 색을 나타내는 값 (0 ≤ mColor ≤ 1)

void paint(int mR, int mC, int mColor)

(mR, mC)에 색이 mColor인 페인트칠을 한다.

 

페인트칠하는 과정은 본문 설명을 참조하라.

 

Parameters
  mR : 페인트칠을 하는 셀의 행의 위치 (0 ≤ mR ≤ N - 1)
  mC : 페인트칠을 하는 셀의 열의 위치 (0 ≤ mC ≤ N - 1)
  mColor : 페인트칠의 색을 나타내는 값 (0 ≤ mColor ≤ 1)

int getColor(int mR, int mC)

(mR, mC)의 색의 값을 반환한다.

 

Parameters
  mR : 색의 값을 알고자 하는 셀의 행의 위치 (0 ≤ mR ≤ N - 1)
  mC : 색의 값을 알고자 하는 셀의 열의 위치 (0 ≤ mC ≤ N - 1)

Returns
  (mR, mC)의 색의 값
