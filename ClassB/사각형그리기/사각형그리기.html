[문제 설명]

크기가 N*N인 격자판에 직사각형들을 그리려고 한다.

 

[Fig. 1]은 N이 10인 격자판을 보여준다.

[Fig. 1]

 

격자판에 하나의 직사각형을 그리기 위해 다음 값들이 주어진다.

1. 사각형의 ID : 그려질 직사각형의 고유번호로 모든 직사각형들은 다른 고유번호를 가지고 있다.

2. 사각형의 위치 : 격자판 내에서 그려질 직사각형의 상좌 꼭짓점의 위치 (행, 열)로 주어진다.

3. 사각형의 크기 : 사각형이 차지하는 셀의 개수를 기준으로 세로 길이와 가로 길이로 주어진다.
   ※ 사각형의 최대 크기는 테스트 케이스마다 ‘L’ 값으로 주어진다.

 

위치가 (3, 1) 그리고 세로 길이 3, 가로 길이가 6인 사각형은 [Fig. 2](a)와 같이 그려진다.

[Fig. 2](b)는 위치가 (0, 2)이고, 세로 길이 8, 가로 길이 2인 사각형을 그린 경우이다.

[Fig. 2]

 

“사각형 그룹”이란 하나 이상의 사각형들의 모임으로 다음 조건에 따라 형성되거나 합쳐진다.

1. 그려진 하나의 사각형은 하나의 사각형 그룹이 된다.

2. 사각형 그룹 내의 하나의 사각형이 다른 사각형 그룹 내의 사각형 하나와 겹치는 경우,

   두 그룹은 하나로 합쳐진다.

 

사각형이 그려지면 위 조건에 의해 그려진 사각형이 하나의 사각형 그룹이 되거나

다른 그룹들과 합쳐지게 된다.

 

두 개의 사각형이 겹치는지 여부는 그러진 영역 내에 공유하는 셀이 하나 이상 존재하는지 여부로 판단한다.

[Fig. 3](a)는 빨간색, 파란색, 노란색 사각형들이 순서대로 그려진 것을 보여준다.

빨간색 사각형과 파란색 사각형은 서로 하나의 셀을 공유하므로 겹친 상태이다.

파란색과 노란색 사각형도 서로 4개의 셀들을 공유하므로 겹친 상태이다.

[Fig. 3](b)에서 3개의 사각형들은 서로 공유하는 셀들이 없으므로 서로 겹치지 않았다.

따라서 [Fig. 3](a)에는 하나의 사각형 그룹이 있고, [Fig. 3](b)에는 세 개의 사각형 그룹이 존재하게 된다.

 

[Fig. 3]

 

[Fig. 4]는 여러 개의 사각형 그룹들이 하나로 합쳐지는 예를 보여준다.

[Fig. 4](a) 상태에서 [Fig. 4](b)와 같이 사각형이 하나 추가 된다면 다른 그룹들과 겹치게 되어

겹쳐진 사각형 그룹들은 하나의 그룹으로 합쳐진다.

[Fig. 4](a)에는 두개의 사각형 그룹이 있고 각각의 그룹은 사각형을 3개씩 가지고 있다.

[Fig. 4](b)에는 하나의 사각형 그룹이 존재하며 그룹 내의 사각형의 개수는 7개이다.

 

[Fig. 4]

 

 

주어진 격자 내에 사각형들을 그리면서 사각형 그룹들을 구분하고, 그룹 내의 사각형의 개수를 얻는 프로그램을 작성해야 한다.

 

아래 API 설명을 참조하여 각 함수를 구현하라.

※ 아래 함수 signature는 C/C++에 대한 것으로 다른 언어에 대해서는 제공되는 Main과 User Code를 참고하라.

 

아래는 User Code 부분에 작성해야 하는 API 의 설명이다.

 

void init(int L, int N)

 

각 테스트 케이스의 처음에 호출된다.


그려질 사각형의 최대 세로 길이와 최대 가로 길이가 L로 주어진다.

사각형을 그릴 격자판의 크기가 N*N으로 주어진다.

 

격자판의 크기 N은 L * 50보다 작거나 같다.

 

격자판은 비어있다.

 

__Parameters

_____L : 사각형의 최대크기 (5 ≤ L ≤ 100)

_____N : 격자판의 크기 (10 ≤ N ≤ L * 50)

 

int draw(int mID, int mRow, int mCol, int mHeight, int mWidth)

 

격자판에 그릴 사각형의 정보가 주어진다.

사각형을 그린 후, 그 사각형이 포함된 그룹 내의 사각형의 개수를 반환한다.

그려질 사각형의 고유번호는 mID이고, 위치는 (mRow, mCol)이다.

사각형의 크기는 세로 길이 mHeight, 가로 길이 mWidth이다.

 

같은 위치에 사각형이 중복되어 그려지지 않는다. 다시말해 사각형의 상좌 꼭짓점의 위치가 같은 경우는 없다.

 

이미 그려진 사각형과 동일한 사각형의 고유번호는 주어지지 않는다.

그려질 사각형은 격자판을 벗어나지 않음을 보장한다.

 

__Parameters

____ mID : 그려질 사각형의 고유번호 (1 ≤ mID ≤ 1,000,000,000)

____ mRow : 그려질 사각형의 행번호 (0 ≤ mRow ≤ N-1)

____ mCol : 그려질 사각형의 열번호 (0 ≤ mCol ≤ N-1)

____ mHeight : 그려질 사각형의 세로 길이 (2 ≤ mHeight ≤ L)

____ mWidth : 그려질 사각형의 가로 길이 (2 ≤ mWidth ≤ L)

 

__Returns

_____r주어진 사각형이 포함된 그룹 내의 사각형의 개수

 

 

int getRectCount(int mID)

 

고유번호가 mID인 사각형이 속한 그룹 내의 사각형의 개수를 반환한다.

mID인 사각형이 존재하지 않는 경우, 0을 반환한다.

 

__Parameters

____ mID : 사각형의 고유번호 (1 ≤ mID ≤ 1,000,000,000)

 

__Returns

_____mID인 사각형을 포함하는 그룹 내의 사각형의 개수
_____mID인 사각형이 없는 경우 0

 

int countGroup()

 

격자판 내에 존재하는 사각형 그룹의 개수를 반환한다.

 

__Returns

_____r격자판 내에 존재하는 사각형 그룹의 개수
